What is this?

Inspired by discussions of cardinality, this program takes an image and traces it along three different space filling curves.

A more in-depth writeup can be found in the GitHub repository, linked here.

How does it work?

Start by uploading any image into the input box. You can also click the "Use Sample Image" button to load a sample image. It will be automatically resized to 32x32. The best images to use should have distinct groups of color.

Then click the "Start Unrolling" button to fill out the lower three images. You can right-click them to save them on your computer, if you wish.
